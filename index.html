Yüz Tespiti ile Öğrenci Yoklama Sistemi

Geleneksel yoklama yöntemlerini dijitalleştirerek zaman kayıplarını sıfıra indiren bir masaüstü yazılımı. Sınıf kamerası üzerinden alınan canlı görüntü akışında yüz tespiti ve öğrenci tanıma işlemleri gerçekleştirilerek, SQL Server veritabanı entegrasyonuyla yoklama verileri otomatik olarak sisteme işlenir.

  • C# ve .NET Framework kullanılarak Windows Form ortamında tasarlandı.
  • Görüntü işleme kütüphanesi olarak OpenCV tabanlı EmguCV kullanıldı.
  • Öğrenci verileri, ders programları ve yoklama çıktıları SQL Server üzerinde ilişkisel olarak tutuldu.
